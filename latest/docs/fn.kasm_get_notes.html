kasm_get_notes - returns a Ableton MIDI Generator array of Note data JS object

For drum loops (drummer) and tape loops (looper), this function simulates stepping through the pattern to generate notes for the full clip duration. The pattern is repeated to fill the specified number of steps.

ยงArguments

  • rate_ms - Rate in milliseconds (used for timing calculations)
  • inlet_0_note - Base MIDI note number
  • inlet_1_semitone - Semitone offset
  • inlet_2_velocity - Velocity
  • inlet_3_enc1 - Encoder 1 (controls pattern parameters)
  • inlet_4_enc2 - Encoder 2 (controls pattern parameters)
  • inlet_5_selector - Pattern/emanator selector
  • inlet_6 - Number of steps per pattern (default 16 if 0)
  • inlet_7 - Number of pattern repetitions (default 1 if 0)
  • inlet_8 - Reserved
