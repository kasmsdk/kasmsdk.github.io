§Kasm Rust SDK for Ableton Live Integration

This crate provides a bridge between web-based MIDI instruments and Ableton Live, allowing for seamless integration of custom MIDI processing and generation logic written in Rust and compiled to WebAssembly (WASM).

The primary goal of this SDK is to enable developers to create sophisticated MIDI devices that can run in a web environment (e.g., within a Max for Live device) and interact with Ableton Live in real-time.

This documentation is automatically generated by running npm run doc, which uses cargo doc to create this sutdoc HTML output.

§Core Concepts

The SDK is built around the idea of “emanators” – modules that generate or transform MIDI data. These can range from simple arpeggiators and LFOs to complex pattern generators and algorithmic composition tools.

Key features include:

  • MIDI Communication: Functions for sending and receiving MIDI notes, CC messages, and pitch bend data.
  • Polyphony Management: Tools to handle polyphonic voice allocation and prevent note flooding.
  • Pattern Generation: A framework for creating and managing MIDI patterns, including arpeggiators, drum sequencers, and melodic generators.
  • Ableton Integration: Support for generating MIDI clips in Ableton Live’s format, allowing for the creation of MIDI data that can be directly used in a project.
  • Real-time Control: Integration with Max for Live UI elements, allowing for real-time control of emanator parameters.

§Examples

§Sending a MIDI note

send_note(60, 127, 0, 500, 64); // Middle C, full velocity, delay 0ms, duration 500ms, centered pan

§Sending a Control Change message

send_cc(10, 127, 0); // Set CC 10 to max value immediately

§Using the bang function for pattern generation

bang(60, 0, 127, 0, 0, 1); // Trigger pattern 1 with Middle C, full velocity

§Features Overview

  • MIDI Messaging: Send raw MIDI messages, notes, CC, and pitch bend.
  • Polyphony Management: Track and manage active notes for polyphony.
  • Pattern Generation: Generate MIDI patterns with various algorithms.
  • Time Management: Synchronized timing for note scheduling and playback.
  • Ableton Integration: Collect and format notes for Ableton MIDI Generator.

§Notes

  • This SDK is designed to be used in a WASM environment, such as within Max4Live devices.
  • For detailed documentation on each function and structure, please refer to the Rustdoc comments in the source code.
